HTML: HyperText Markup Language

HTML is the backbone of web content. It uses tags to structure and format documents.

"HTML is like the skeleton of a webpage, providing the structure that CSS can style." - Web Developer Proverb

Common HTML Tags

Tag Description
<div> Defines a division or section
<p> Represents a paragraph
<a> Creates hyperlinks
<img> Embeds images
